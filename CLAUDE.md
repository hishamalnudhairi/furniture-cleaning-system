# CLAUDE.md — قواعد المشروع

> هذا الملف يوجّه Claude Code أثناء العمل على المشروع. اقرأه قبل تنفيذ أي مهمة.

## نبذة عن المشروع
نظام ويب (MVP) لإدارة **مغسلة كنب وزوالي وسجاد**. يخدم:
- **العمال**: واجهة بسيطة جدًا لتسجيل ومتابعة الطلبات.
- **المحاسب**: مراجعة الطلبات، تحديد السعر، إصدار الفواتير.
- **العميل**: صفحة خارجية لطلب الخدمة بدون فاتورة.
- **السائقون**: استلام/تسليم الطلبات ومحاسبتهم.

## التقنيات (Stack)
- **Backend**: Laravel (أحدث إصدار LTS مستقر).
- **Database**: MySQL.
- **Views**: Blade.
- **Styling**: Tailwind CSS.
- **اللغات**: العربية (افتراضي، RTL) والإنجليزية (LTR) عبر نظام الترجمة `lang/`.

## قواعد العمل مع Claude Code (مهمة جدًا)
1. **لا تنفّذ أوامر خطيرة** (حذف، تهيئة، أوامر نظام واسعة الأثر).
2. **لا تحذف ملفات** للمستخدم إطلاقًا دون طلب صريح ومراجعة.
3. **لا تبنِ النظام كله مرة واحدة**. نفّذ مرحلة واحدة في كل مرة حسب `PLAN.md`.
4. **اطلب التأكيد** قبل: إنشاء مشروع Laravel، تثبيت الحزم، تشغيل migrations، أي أمر يعدّل النظام.
5. **اشرح بإيجاز** قبل كل خطوة ما الذي ستفعله ولماذا.
6. المستخدم **مبتدئ** — استخدم لغة واضحة وخطوات صغيرة قابلة للمتابعة.
7. بعد كل مهمة، قدّم **ملخصًا مختصرًا** لما تم.

## مبادئ التصميم
- **البساطة أولًا**: الواجهة موجهة لعمّال قليلي الخبرة — أزرار كبيرة، نص واضح، أقل عدد خطوات.
- **جمال هادئ**: تصميم نظيف بـ Tailwind، ألوان قابلة للتخصيص من الإعدادات.
- **RTL أولًا**: التصميم يدعم العربية بشكل صحيح، والإنجليزية ثانويًا.
- **الموبايل أولًا**: معظم المستخدمين على الجوال.

## قواعد الكود
- اتبع اصطلاحات Laravel الرسمية (Controllers رفيعة، منطق في Services/Actions عند الحاجة).
- أسماء واضحة بالإنجليزية في الكود، النصوص الظاهرة للمستخدم عبر ملفات الترجمة.
- لا نصوص ثابتة (hardcoded) في الـ Blade — استخدم `__('...')`.
- التحقق من المدخلات عبر Form Requests.
- لا تكتب أسرارًا (مفاتيح، كلمات مرور) في الكود — استخدم `.env`.

## حالات الطلب (Order Status)
- `pending_review` — بانتظار المراجعة (الافتراضي عند طلب العميل).
- `in_progress` — قيد التنفيذ.
- `priced` — تم تحديد السعر.
- `invoiced` — تم إصدار الفاتورة.
- `out_for_delivery` — خرج للتوصيل.
- `completed` — مكتمل.
- `cancelled` — ملغي.

## ما يجب عدم فعله الآن
- ❌ لا تُنشئ مشروع Laravel بعد.
- ❌ لا تثبّت أي حزم بعد.
- ❌ لا تشغّل أي migration بعد.
